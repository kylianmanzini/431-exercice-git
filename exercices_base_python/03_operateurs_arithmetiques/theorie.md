# 03 — Opérateurs arithmétiques

Concepts clés:
- `+`, `-`, `*`, `/` (division flottante), `//` (division entière), `%` (modulo), `**` (puissance).

Exemples:
```python
9 / 4   # 2.25
9 // 4  # 2
9 % 4   # 1
2 ** 3  # 8
```

Conseils:
- Attention à la division entière avec nombres négatifs.
- Utiliser `round(x, ndigits)` pour l’affichage.
