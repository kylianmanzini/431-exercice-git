# 04 — Comparaisons et booléens

Concepts clés:
- Comparaisons: `==`, `!=`, `>`, `<`, `>=`, `<=`.
- Logique: `and`, `or`, `not` pour composer des conditions.

Exemples:
```python
(10 > 5) and (5 > 0)  # True
not (3 == 4)          # True
```

Conseils:
- Comparer des types compatibles.
- Parenthéser les expressions complexes pour la lisibilité.
