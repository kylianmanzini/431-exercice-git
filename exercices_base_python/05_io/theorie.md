# 05 — Entrées / sorties

Concepts clés:
- `input()` lit une chaîne depuis la console.
- `print()` affiche des valeurs, séparées par `sep` et terminées par `end`.

Exemples:
```python
nom = input("Votre nom: ")
age = int(input("Votre âge: "))
print("Bonjour", nom, "— âge:", age)
```

Conseils:
- Toujours convertir les entrées vers le type voulu.
- Valider et gérer les erreurs éventuelles.
