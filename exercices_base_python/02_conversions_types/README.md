# 02 — Conversions de type

Objectifs:
- Convertir entre `int`, `float`, `str` et structures.

Théorie:
- Casting: `int(3.7) -> 3`, `float("2.0") -> 2.0`, `str(42) -> "42"`.
- Attention aux erreurs: `int("abc")` lève `ValueError`.
