# 01 — Variables

Objectifs:
- Comprendre et déclarer des variables des types `int`, `float`, `str`, `list`, `tuple`, `dict`.
- Savoir lire et afficher leurs valeurs.

Théorie:
- Python infère le type à l’affectation: `x = 3`, `y = 2.5`, `nom = "Alice"`.
- Collections: `list` mutable, `tuple` immuable, `dict` paires clé/valeur.

Consignes:
- Déclarer une variable de chaque type demandé.
- Modifier la liste, lire un élément du tuple, ajouter une clé au dict.
- Afficher les valeurs et types avec `print()` et `type()`.
