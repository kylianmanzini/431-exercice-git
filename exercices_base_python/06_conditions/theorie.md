# 06 — Conditions

Concepts clés:
- Structure conditionnelle: `if`, `elif`, `else`.
- Les blocs s’exécutent selon la vérité des conditions.

Exemples:
```python
note = 5.2
if note < 4: print("insuffisant")
elif note < 5.5: print("moyen")
else: print("bien")
```

Conseils:
- Ordre des conditions: du plus spécifique au plus général.
- Préférer des comparaisons claires et lisibles.
